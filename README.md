# Parallel Example: Style Fidelity Grid

This packet models how an image-generation and art-style interest could become a research journal and paper-lite draft. It is adjacent to art-style control work, but it does not describe the student's actual project.

## Core idea

The imagined Space is a **Style Fidelity Grid**. A user enters one visual prompt and runs it across several style settings or models. The project asks whether "style control" changes deep visual choices or mostly adds surface markers.

## Research question

When the same prompts are run through general-purpose and style-tuned image models, how much of style control is real and how much is just surface-level style vocabulary?

## Space idea

Inputs:

- Scene prompt
- Style: anime, surrealist, watercolor, noir, or realistic
- Model type: general-purpose or style-tuned
- Optional negative prompt

Outputs:

- Image or prompt-output comparison grid
- Style fidelity checklist
- Notes on content preservation and flexibility
- Reflection prompt asking what changed besides obvious style markers

## What this packet demonstrates

- How to turn image-model curation into research evidence
- How to test style fidelity with repeated prompts
- How to connect classroom work to style transfer, LoRA, and prompt-engineering research
- How to write about specialization tradeoffs
